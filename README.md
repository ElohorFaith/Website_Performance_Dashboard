2. DAX
3. Data preparation

1. Average Bounce Rate by Visitor Type 
2. Average Conversion Rate by Visitor Type 
3. Average Conversion Rate by City
4. Top 100 visitors
5. Card for Average Page views
6. Card for Average Session Duration
7. Card for Average Bounce Rate
8. Card for Average Conversion Rate
