# Website_Performance_Dashboard
This project analyses website engagement and conversion rate using Power BI. 

# Tools Used
1. Power BI
2. DAX
3. Data preparation

# Key Metrics
1. Average Page views
2. Average Bounce rate
3. Averation Session duration
4. Average Conversion rate

# Key Insights
1. Bounce rate remains around 50%, indicating half of visitors exit after a single page.
2. Conversion rate averages 5%, suggesting opportunity for optimization.
3. Session duration varies by visitor type and exit page.
4. High-converting visitors show stronger engagement patterns.
